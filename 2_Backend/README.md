## 🎯 핵심 개념

### API 설계
- **REST 원칙**: 자원 중심 설계
- **HTTP 메서드**: GET, POST, PUT, DELETE 적절한 활용
- **상태 코드**: 명확한 응답 상태 전달
- **버전 관리**: API 하위 호환성 유지

### 데이터베이스
- **정규화**: 중복 제거 및 무결성 보장
- **인덱싱**: 쿼리 성능 최적화
- **트랜잭션**: ACID 원칙 준수
- **마이그레이션**: 스키마 버전 관리
