## 🎯 핵심 개념

### Docker
- **컨테이너화**: 애플리케이션과 의존성을 패키징
- **이미지 레이어**: 효율적인 저장 및 배포
- **볼륨 관리**: 데이터 영속성 보장
- **네트워킹**: 컨테이너 간 통신

### CI/CD
- **지속적 통합**: 코드 변경사항 자동 통합
- **지속적 배포**: 자동화된 배포 파이프라인
- **Jenkins Pipeline**: 코드로 정의하는 빌드 프로세스
