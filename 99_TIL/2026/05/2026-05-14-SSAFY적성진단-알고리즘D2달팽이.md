## 오늘 배운 것

### [Logic] 달팽이 나선형 탐색 — 방향 배열 + 전환 조건

N×N 행렬을 나선형으로 채우는 문제다.  
핵심 아이디어는 **방향을 배열로 관리하고, 막히면 오른쪽으로 90도 전환**하는 것.

```python
# 방향: 우 → 하 → 좌 → 상 순서
dx = [0, 1, 0, -1]  # 행 이동
dy = [1, 0, -1, 0]  # 열 이동

# 다음 칸이 범위 초과 또는 이미 채워졌으면 방향 전환
if nx < 0 or nx >= N or ny < 0 or ny >= N or matrix[nx][ny] != 0:
    direction = (direction + 1) % 4
```

`% 4`로 방향 인덱스를 순환시키는 게 핵심.  
4가 되면 다시 0(우)으로 돌아온다.

그래도 받은 코드를 그냥 제출하지 않고, **왜 이렇게 되는지 이해하면서** 내 손으로 다시 작성했다. `dx`, `dy`가 왜 저 순서인지, `% 4`가 왜 필요한지, 조건 검사를 왜 숫자 채우기 *이후에* 하는지.
