## 1강. 디지털 트랜스포메이션

디지털화(Digitalization)와 디지털 트랜스포메이션(DT)은 다르다. 단순히 아날로그를 디지털로 바꾸는 것이 디지털화라면, DT는 데이터를 기반으로 **비즈니스 모델 자체를 재설계**하는 구조적 변화다.

- 4차 산업혁명의 핵심 동력: AI, IoT, 클라우드, 빅데이터
- DT의 본질은 기술 도입이 아니라 데이터를 중심으로 의사결정 방식을 바꾸는 것
- 기업이 DT에 실패하는 주된 이유: 기술 도입에만 집중하고 조직·문화 변화를 간과

---

## 2강. 빅데이터의 이해

빅데이터를 정의하는 3V에서 출발해 5V로 확장된다.

| V | 의미 |
|---|------|
| Volume | 데이터의 규모 (TB~PB 단위) |
| Velocity | 데이터 생성·처리 속도 (실시간) |
| Variety | 정형·반정형·비정형 데이터의 다양성 |
| Value | 분석을 통해 추출하는 비즈니스 가치 |
| Veracity | 데이터의 신뢰성과 정확성 |

빅데이터의 등장 배경은 소셜미디어, IoT 기기 확산, 클라우드 인프라의 보편화로 데이터 생성량이 폭발적으로 증가한 것. 전 세계 데이터의 90%가 최근 2년 안에 생성됐다는 것이 이를 설명한다.

---

## 3강. 빅데이터 비즈니스 모델과 가치

빅데이터가 만들어내는 비즈니스 가치 유형:
- **투명성 향상**: 데이터를 공유해 조직 내 의사결정 속도 향상
- **성능 최적화**: 실시간 데이터로 운영 효율 개선
- **개인화 서비스**: 고객 행동 분석 기반 맞춤 추천

실제 사례:
- **넷플릭스**: 시청 이력·평점 데이터로 콘텐츠 추천 및 오리지널 제작 판단
- **아마존**: 구매 패턴 분석으로 재고 최적화 및 구매 예측
- **의료 분야**: 환자 데이터 분석으로 질병 조기 진단

데이터 기반 의사결정(DDDM, Data-Driven Decision Making)이 직관 기반 의사결정을 대체하는 흐름.

---

## 4강. 빅데이터 분석 기법과 데이터 마이닝

데이터 마이닝의 주요 기법:

| 기법 | 설명 | 예시 |
|------|------|------|
| 분류(Classification) | 데이터를 미리 정의된 범주로 구분 | 스팸 메일 분류 |
| 군집화(Clustering) | 유사한 데이터끼리 자동 그룹화 | 고객 세분화 |
| 연관 규칙(Association Rule) | 항목 간 동시 발생 패턴 탐색 | 장바구니 분석 |
| 회귀분석(Regression) | 변수 간 관계로 수치 예측 | 주가 예측 |

데이터 전처리가 전체 분석 시간의 70~80%를 차지한다는 점이 핵심. 결측값 처리, 이상치 제거, 정규화 과정 없이는 분석 결과를 신뢰할 수 없다.

## 5강. 기계학습의 이해

기계학습(Machine Learning)은 명시적인 규칙 프로그래밍 없이 **데이터로부터 패턴을 스스로 학습**하는 기술.

학습 방식 분류:

| 구분 | 특징 | 예시 |
|------|------|------|
| 지도학습(Supervised) | 정답(레이블) 있는 데이터로 학습 | 이메일 스팸 분류, 이미지 인식 |
| 비지도학습(Unsupervised) | 레이블 없이 패턴 스스로 발견 | 고객 군집화, 이상 탐지 |
| 강화학습(Reinforcement) | 보상 신호로 행동 전략 학습 | 게임 AI, 로봇 제어 |

과적합(Overfitting): 훈련 데이터에만 지나치게 맞춰져 새 데이터에 대한 예측력이 떨어지는 현상. 데이터를 훈련/검증/테스트 세트로 분리해 관리하는 이유.

## 6강. 빅데이터 분석 과정과 기술

**분석 프로세스 6단계:**

```
문제 정의 → 데이터 수집 → 데이터 전처리 → 분석·모델링 → 시각화 → 인사이트 도출
```

주요 빅데이터 처리 기술:
- **Hadoop**: 대용량 데이터를 분산 저장·처리하는 오픈소스 프레임워크. HDFS(저장) + MapReduce(처리)
- **Spark**: Hadoop보다 최대 100배 빠른 인메모리 처리. 실시간 분석에 강점
- **NoSQL DB**: MongoDB, Cassandra 등 비정형 데이터를 유연하게 저장

시각화 도구: Tableau, Power BI, Python(matplotlib, seaborn)

## 핵심 인사이트

1. **빅데이터의 가치는 데이터 자체가 아니라 분석과 해석에 있다** — 아무리 많은 데이터도 목적 없이는 노이즈에 불과하다
2. **전처리가 분석의 핵심이다** — 좋은 모델보다 좋은 데이터가 우선이다
3. **DT는 기술 도입이 아니라 의사결정 방식의 전환이다** — 도구를 바꾸는 게 아니라 사고방식을 바꾸는 것
