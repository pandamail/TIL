### 1. 시나리오 재시작 시 바이탈 미표시 문제 해결
- **문제 상황**:
  - SimMonitor 재시작 없이 시나리오 반복 시 일부 바이탈이 "--"로 표시
  - 이전 세션에서 변하지 않았던 값들만 미표시되는 패턴 발견

- **원인 분석**:
  ```
  1. 세션 시작 시 Vitals.Reset() 호출 → 모든 바이탈 null로 초기화
  2. LLEAP은 효율성을 위해 변경된 파라미터만 전송
  3. 이전과 같은 값의 바이탈은 전송 안 함 → null 유지 → "--" 표시
  ```

- **해결 방법**:
  ```csharp
  // ResetSessionState()에서 Vitals.Reset() 호출 제거
  // 이전 바이탈 값을 유지하면서 LLEAP이 전송한 값만 업데이트
  ```
