## 핵심 개념 정리

### 차트

**Q22. 보조 축 / 데이터 레이블**
- 수량 계열이 보조 축으로 지정되면 오른쪽 Y축에 표시됨
- 데이터 레이블 위치는 차트에서 직접 확인 (위쪽/가운데/바깥쪽 구분)

**Q34. 원형 차트**
- 원형 차트는 **축이 없음**
- 원형 대 가로 막대형: 작은 값을 **가로 막대형**으로 분리 표시 (누적 아님)
- 3차원 원형 차트도 쪼개진 원형으로 표시 가능

---

### 정렬 / 페이지 설정

**Q23. 정렬**
- 숨겨진 행/열은 **정렬에 포함되지 않음** ← 암기

**Q25. 페이지 설정 - 자동 맞춤**
- 용지 너비/높이를 각각 1로 지정 → 여러 페이지가 **한 페이지**에 인쇄됨
- 머리글/바닥글 여백 단위: **인치** (mm 아님)

---

### 조건부 서식

**Q26. 수식으로 조건 지정**
- 특정 셀 클릭 시 **절대 참조**로 작성됨
- ④ 이동 옵션으로 조건부 서식 셀 찾기 가능 → 맞는 설명 (정답)

> 헷갈리면 직접 엑셀에서 확인해볼 것

---

### 함수 / 수식

**Q27. RANK + 배열 수식 (동점자 처리)**

```excel
{=RANK($F2,$F$2:$F$10)+SUM(($F$2:$F$10=$F2)*($B$2:$B$10>$B2))}
```

- 해석: 총점이 같은 사람 중 국어 점수가 나보다 높은 사람 수를 더함

**Q28. 사용자 지정 표시 형식**
- 조건 3개 + 소수 첫째 자리: `[빨강][>=200]#,##0.0;[파랑][>=100]#,##0.0;#,##0.0`
- 세미콜론 3개 → 4번째는 텍스트용 / 조건 3개면 세미콜론 2개

---

### VBA / 매크로

**Q29. A열 굵게 설정 매크로**
- `Range("1:1")` → 1행 전체 선택 (A열 아님) ← **틀린 코드**
- A열 굵게: `Columns("A")`, `Columns(1)`, `Range("A:A")` 모두 가능

**Q33. Worksheet_Activate 이벤트**
- 워크시트 **활성화될 때** 실행
- `Key1=Range("A2")` → A2 기준 오름차순 정렬

**Q38. 매크로 바로가기 키**
- **Ctrl + 영문자** 조합만 가능
- Alt는 매크로 바로가기 키에 사용 불가

---

### 틀 고정 / 시나리오

**Q30. 틀 고정**
- 틀 고정 구분선은 **마우스로 이동 불가**
- 창 분할 구분선은 이동 가능 (틀 고정과 다름)

**Q32. 시나리오 변경 셀**
- 변경 셀 최대 **32개**
- 변경 금지 = 편집/삭제 방지 (추가 방지 아님)

---

### 데이터 통합 / 피벗 테이블

**Q36. 데이터 통합**
- 원본 데이터 연결은 같은 워크시트가 아니어도 적용 가능
- "같은 경우에만" → 틀린 설명

**Q37. 피벗 테이블**
- 원본 데이터 변경돼도 **자동 반영 안 됨**
- **새로 고침** 해야 반영됨 ← 무조건 암기
